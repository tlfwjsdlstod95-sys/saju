# The Scientific Oracle — 정통 만세력 AI 사주

천문 알고리즘 기반 **초정밀 만세력** 엔진과 Next.js 풀스택 웹앱. 경도 보정·균시차·서머타임·야자시/조자시까지 반영해 "가장 정확한 정통 명리"를 지향합니다.

## 빠른 시작

```bash
npm install
npm run dev        # http://localhost:3000
```

```bash
npm run test:engine   # 엔진 검증 (앵커·년주·입춘 경계·야자시)
npm run build         # 프로덕션 빌드
```

## 프로젝트 구조

```
app/
  page.tsx              입력 폼 + 결과 대시보드 (클라이언트)
  api/saju/route.ts     POST /api/saju → 만세력 계산 API
  layout.tsx, globals.css
lib/saju/               ★ 만세력 엔진 (의존성 없는 순수 TS)
  astro.ts              율리우스일, 태양 황경, 균시차, ΔT (Meeus 알고리즘)
  solarTerms.ts         절기 황경 → 월지/입춘(년) 경계 판정
  constants.ts          천간/지지/오행/지장간/서머타임/표준자오선 테이블
  elements.ts           십신 판정, 오행 집계, 일간 강약, 성향 생성
  index.ts              computeSaju() 메인 오케스트레이션
  types.ts              타입 정의
scripts/test-engine.ts  검증 스크립트
```

## 엔진 동작 원리

1. **시간대 보정** — 출생 시각(시계) → UTC. 표준자오선 변경 이력(127.5°E/135°E)과 서머타임(1948–88) 자동 반영.
2. **진태양시(眞太陽時)** — UTC + 경도/360일 + 균시차. 일주·시주는 이 국소 겉보기 태양시로 산출 → 야자시(23시)/조자시(0시) 자연 분기.
3. **년주** — 입춘(立春, 태양황경 315°) 순간을 뉴턴법으로 정밀 산출해 경계 판정.
4. **월주** — 출생 순간의 태양 황경으로 월지 직접 결정 + 월두법(五虎遁) 천간.
5. **일주** — 진태양시 달력일의 율리우스일(JDN) → 60갑자. 시두법(五鼠遁)으로 시간 천간.
6. **해석** — 지장간·오행 분포·십신·일간 강약 → 현대적 성향 카드.
7. **대운/세운** — 양남음녀 순행·음남양녀 역행, 대운수(출생~절기 일수÷3), 월주 기준 60갑자 순/역 배열. 일간 강약 기반 길흉 점수(−100~100)로 운세 흐름 그래프 산출.
8. **공유 카드** — 일간 오행 + 우세 십신으로 MBTI식 유형(예: "타오르는 크리에이터") 생성, 1080×1350 캔버스 PNG로 인스타/X 공유 (Web Share API · 이미지 저장 · 문구 복사).
9. **명식 고도화** — 십이운성(十二運星), 공망(空亡), 신살(도화·역마·화개·천을귀인·문창귀인·양인·괴강·백호) 자동 판별.
10. **사주 풀이 ('선배 톤')** — 한 줄 정의 + 10단계(①당신이라는 사람 ②타고난 무기 ③발목 잡는 것 ④올해 ⑤연애 ⑥돈·진로 ⑦몸·에너지 ⑧사람복 ⑨큰 그림 ⑩마지막으로). 따뜻하지만 직설적, 20대 타깃, 실제 명식·올해 세운·나이 반영. 펼쳐진 긴 글 레이아웃. 프리미엄 업셀 + 궁합 크로스셀.

11. **AI 심층 풀이 (LLM 레이어, 스트리밍)** — `/api/reading`. 결정론 엔진이 계산한 명식(사주팔자·십신·오행·신살·대운·세운)을 **사실로 고정**해 LLM에 넘기고, '선배 톤' 프롬프트로 10단계를 개인화해 다시 씁니다. 연산은 엔진, 문장만 LLM. 응답은 `@@키@@` 마커 텍스트로 **스트리밍**되어, 섹션별로 한 글자씩 써지는 타이핑 연출(깜빡이는 커서)로 렌더링됩니다. 키가 없으면 규칙 기반 기본 풀이로 무리 없이 폴백.

12. **신뢰 연출** — 히어로 아래 검증 배지(VSOP87·KASI 기준·절기/균시차/경도/야자시 보정·정밀도 99.999%), 분석 시 천문 계산 단계가 순차 진행되는 정밀 분석 오버레이(회전 궤도 + 진태양시→경도보정→절기→명식→분석).
13. **풀이 공유 카드(ReadingCard)** — 한 줄 정의 + 핵심 섹션 티저를 담은 1080×1350 골드 명조 카드. AI/기본 풀이 모두 지원, 저장·공유·문구 복사.
14. **일간 물상(物象) 유형 카드** — 정통 명리의 일간 물상으로 본질을 표현(병화=☀️한낮의 태양, 계수=💧비와 이슬, 갑목=🌲큰 나무, 신금=💎보석…). 강약·십신으로 수식어/트레잇 부여. 한자 부담 없이 가독성·공유성 높인 ShareCard.
15. **궁합 AI 스트리밍** — `/api/gunghap-reading`. 두 명식+상성 점수를 근거로 '연애 고수 선배' 톤 궁합 풀이를 6단계로 타이핑 스트리밍.
16. **과금 잠금(Paywall)** — AI 심층 풀이·AI 궁합·정밀 리포트를 프리미엄으로 잠금. 결제 모달(₩9,900, 데모) → `localStorage` 영속 해제 → 잠금 풀린 기능 자동 실행. 무료(규칙 기반 풀이·명식·대운·궁합 점수·물상 카드)는 그대로 열람.
17. **내 사주 보관함** — 나·가족·친구 사주를 `localStorage`에 저장(물상·일주 요약 포함). 칩 클릭으로 즉시 재분석, 궁합 페이지에서 두 사람을 보관함에서 바로 불러오기.

**정확도**: 절기 시각이 한국천문연구원 공표값과 **최대 1분 이내** 일치 (VSOP87 정밀 태양황경). 일주 60갑자 국제표준 보정.

테마: 신비로운 사주 감성 — 딥 퍼플 + 골드, 명조체(Nanum Myeongjo) 제목.

## AI 심층 풀이 설정 (선택)

```bash
cp .env.example .env.local   # ANTHROPIC_API_KEY 채우기
npm run dev
```

키를 넣으면 결과 화면의 "🔮 AI로 다시 풀기" 버튼이 활성화됩니다. 키가 없어도 규칙 기반 풀이·명식·대운·궁합·공유카드는 전부 그대로 동작합니다. 모델은 기본 `claude-sonnet-4-6`(환경변수 `SAJU_MODEL`로 변경 가능).
11. **궁합(상성)** — `/gunghap`. 천간합·지지 육합/삼합/충·오행 상호보완·강약 균형 가중으로 0~100점 + 항목별 분석. 두 사람 유형·점수 링·항목 막대를 담은 궁합 공유 카드(PNG)까지. 개인 카드 → 궁합 → 궁합 카드로 바이럴 루프 완성.

## 정확도 기준 (검증됨)

| 항목 | 기준 | 결과 |
| --- | --- | --- |
| 일주 앵커 | 2000-01-01 = 무오일(국제표준 HKO/sxtwl) | ✓ |
| 년주 | 2024 갑진 / 2023 계묘 / 1990 경오 | ✓ |
| 입춘 경계 | 2024-02-03 계묘년 → 02-05 갑진년 | ✓ |
| 야자시 | 23:40 출생 子시·일주 미전진 | ✓ |
| 보정 | 서울 경도 −32분, 균시차 ±16분 | ✓ |

> 태양 황경 정확도 ~0.01° (절기 시각 환산 분 단위). **절기 전후·자정 무렵 경계 출생**은 한국천문연구원(KASI) 교차검증을 권장합니다.

## 배포 (Vercel)

가장 간단한 방법은 Vercel입니다. (무료 시작 · HTTPS 자동 · 스트리밍 AI 지원)

1. **GitHub에 올리기**
   ```bash
   git init && git add -A && git commit -m "init"
   git branch -M main
   git remote add origin https://github.com/<계정>/<레포>.git
   git push -u origin main
   ```
   `.env.local`은 `.gitignore`에 의해 **커밋되지 않습니다**(키 안전).

2. **Vercel Import** — [vercel.com](https://vercel.com) → GitHub 로그인 → 해당 레포 **Import** → Framework는 Next.js 자동 감지 → Deploy.

3. **환경변수 등록** (Vercel → Project → Settings → Environment Variables)

   | 변수 | 설명 |
   | --- | --- |
   | `ANTHROPIC_API_KEY` | AI 심층 풀이·상담·신년 총평 (필수, 없으면 규칙 풀이로 폴백) |
   | `SAJU_MODEL` | (선택) 기본 `claude-sonnet-4-6` |
   | `NEXT_PUBLIC_TOSS_CLIENT_KEY` | 토스 클라이언트 키 (테스트/라이브) |
   | `TOSS_SECRET_KEY` | 토스 시크릿 키 (서버 전용) |
   | `NEXT_PUBLIC_SITE_URL` | 배포 도메인 (예: `https://your-domain.com`) — OG 공유 미리보기용 |

4. **(선택) 커스텀 도메인 연결**, **토스 라이브 키 교체**(사업자등록+가맹점 심사 후).

> 배포 후 PWA 설치·OG 공유 미리보기가 실제로 동작합니다. 환경변수를 바꾸면 **Redeploy** 해야 반영됩니다.
